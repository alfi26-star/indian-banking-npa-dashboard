# Indian Banking NPA & Credit Risk Analysis

## Project Overview
This project analyzes Gross and Net NPA trends in the Indian banking sector from 1996-97 to 2023-24.

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## Features
- NPA trend analysis
- Yearly change analysis
- Rolling average analysis
- Summary statistics
- Worst banking stress year identification

## Key Insights
- NPAs peaked during the late 1990s.
- Banking asset quality improved significantly during 2000-2008.
- Stress increased again after 2014.
- Recent years show improving banking stability.
